## Creating and installing a .deb package

### 1. Clone the repository
```bash
git clone https://github.com/AeEn123/RoExtract
```

### 2. Make `build_deb.sh` executable
```bash
cd RoExtract
chmod +x packages/debian/build_deb.sh
```

### 3. Run the script
```bash
bash packages/debian/build_deb.sh
```

## The package will be built and installed automatically.
