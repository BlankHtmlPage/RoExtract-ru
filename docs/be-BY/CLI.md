# Рэжым каманднага радка (CLI)
> [!NOTE]
> CLI-рэжым прызначаны для дасведчаных карыстальнікаў. Калі вам патрэбны графічны інтэрфейс (GUI), гл. [README.md](README.md)

> [!NOTE]
> CLI-рэжым знаходзіцца ў распрацоўцы. Яго функцыянал абмежаваны, і ў будучыні ўсё можа змяніцца.

Гэты рэжым карысны, калі вы хочаце выкарыстоўваць праграму выдалена (без манітора) і/або калі вы хочаце аўтаматызаваць яе працу.

# Каманды і выкарыстанне
## --help
### Выкарыстанне:
```
./RoExtract --help <каманда>
```
### Апісанне:
Выводзіць даведку са спісам каманд.
### Аргументы:
Аргумент `<каманда>` неабавязковы.
Калі ён пазначаны, выводзіцца даведка для гэтай каманды.

## --list
### Выкарыстанне:
```
./RoExtract --list <катэгорыя>
```
### Апісанне:
Адлюструе файлы ў гэтай катэгорыі.
### Аргументы:
`<катэгорыя>` абавязковая.
`<катэгорыя>` павінна быць адной з наступных: `music`, `sounds`, `images`, `ktx` або `rbxm`.
